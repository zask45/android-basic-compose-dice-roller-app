# Dice Roller App 

A Dice Roller App for Entertaiment. The app will let the user to roll the dice virtually when the 'Roll' button is pressed.

<_Part of Google Developers Android Course Unit 2_>

### Application Architecture
---
- IDE: Android Studio
- Language: Kotlin
- API: API 34
